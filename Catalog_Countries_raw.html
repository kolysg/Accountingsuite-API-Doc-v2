Countries


In order to process a single object one should first read the whole list and then to pick up a certain entry from it.

To get a list of all entries of a kind: Countries:

Example request:

GET https://apps7.accountingsuite.com/a/1100609/odata/standard.odata/Catalog_Countries?$format=json -H "Authorization: Basic ZGVtb0BhY2NvdW50aW5nc3VpdGUuY29tOkRlbW8yMDE2"

Example response:


A single entry can be picked up from the list and processed by its Ref_key (a.k.a. guid). Using a guid in a query string, we command the database to call a certain entry

To get a specific object of a kind Countries:

Example requests:

GET https://apps7.accountingsuite.com/a/1100609/odata/standard.odata/Catalog_Countries(guid'a3e03d56-1ab4-11e3-8b49-bab74d291c34')?$format=json -H "Authorization: Basic ZGVtb0BhY2NvdW50aW5nc3VpdGUuY29tOkRlbW8yMDE2"
